Quick Start


# Clone the repository
git clone https://github.com/ibrahmsql/spiderjs.git
cd spiderjs

# Build the binary
go build -o spiderjs_bin ./cmd/spiderjs

# Run a scan
./spiderjs_bin scan --url https://example.com

Or use our web interface to scan your application.
